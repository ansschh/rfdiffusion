# rfdiffusion

Deployment repo for the OrganoEnzymeGen artificial-metalloenzyme design work — code is developed
locally and pulled onto the Caltech HPC (SLURM) cluster from here.

## Contents
- `cluster/` — SLURM GPU reconnaissance toolkit (find idle GPUs, allowed limits, and real allocation
  wait times before running experiments). See `cluster/README.md`.

More to come: RFdiffusion2 / LigandMPNN / AF2 experiment scripts for the Sub-question B back-end runs.
